# AutoTestMaster

It's a C# project using Selenium WebDriver to test the functionality of the [Automation Form](https://app.cloudqa.io/home/AutomationPracticeForm).
Using C# and Selenium to automatically test fields on the page


## Installation

To use this project, you need to have the following software installed on the system:
- Visual Studio
- Selenium WebDriver
- Chrome Driver

Selenium WebDriver and ChormeDriver can be installed by using NuGet packages:

` Install-Package Selenium.WebDriver ` <br />
` Install-Package Selenium.WebDriver.ChromeDriver `


The program navigates to the Form webpage and fill out the following fields:
- First Name
- Last Name
- Date of birth
- Email
- Mobile
- Country
- Gender
- State
